Book 1

Title Let Us C
Author Yashavant Kanetkar
Publisher BPB Publications
Edition 17th Edition
Language English
Pages 672
Price ₹ 325
Category Programming / C Language

Summary

"Let Us C" is one of the most popular books for learning the C programming language in India. Written by Yashavant Kanetkar, it explains concepts from the very basics to advanced topics in a clear and simple manner. It is widely used in colleges and is excellent for beginners. The book includes many practical programs and exercises to help students understand the fundamentals of programming.

Book 2

Title HTML and CSS: Design and Build Websites
Author Jon Duckett
Publisher Wiley
Edition 1st Edition
Language English
Pages 490
Price ₹ 1,890
Category Web Development

Summary

This is a beautifully designed book that teaches HTML and CSS through a visual approach. Each concept is explained with clear diagrams and examples spread across full pages. Jon Duckett makes web design accessible to complete beginners. The book covers everything from basic HTML structure to layout and styling with CSS. It is ideal for anyone who wants to start building websites.

Book 3

Title Data Structures Using C
Author Reema Thareja
Publisher Oxford University Press
Edition 3rd Edition
Language English
Pages 560
Price ₹ 495
Category Data Structures / Computer Science

Summary

This book by Reema Thareja is a comprehensive guide to data structures using the C language. It covers arrays, stacks, queues, linked lists, trees, graphs, and sorting algorithms with clear explanations and code examples. The book is designed for undergraduate students and follows university syllabi closely, making it very useful for exam preparation.
