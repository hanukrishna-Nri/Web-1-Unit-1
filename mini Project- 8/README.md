MediCare Plus - Healthcare Information System
=============================================

PROJECT OVERVIEW
----------------
MediCare Plus is a beginner-friendly Healthcare Information System developed using HTML. The project simulates a hospital website that provides essential healthcare information, patient services, doctor details, appointment booking facilities, emergency contact information, and educational resources for patients.

The primary objective of this project is to demonstrate the use of various HTML elements such as tables, forms, navigation links, lists, and storage concepts while creating a realistic healthcare portal.


OBJECTIVES
----------
1. Design a structured healthcare information website using HTML.
2. Provide information about hospital services and facilities.
3. Display doctor profiles and availability schedules.
4. Enable patients to submit appointment requests through an online form.
5. Demonstrate Local Storage and Session Storage concepts.
6. Present emergency contact information and patient resources.
7. Improve understanding of webpage layout and user interface design.
